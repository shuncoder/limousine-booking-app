# Web Admin - Limousine Booking

> Dashboard quản trị người dùng, chuyến đi, thống kê, quản lý tài khoản.

## Công nghệ chính
- Next.js 16 (App Router)
- React 19, TypeScript
- Tailwind CSS v4

## Cấu trúc thư mục
- `src/app/(admin)/`: Trang quản trị
- `src/components/`: UI, bảng, biểu đồ
- `src/lib/`: API helper

## Hướng dẫn cài đặt
```bash
cd webadmin
npm install
npm run dev
```

## Chức năng chính
- Đăng nhập admin
- Quản lý user, ride, thống kê
- Kết nối backend qua REST API

# XeAdmin

XeAdmin is an admin dashboard starter built with:

* Next.js 16.x (App Router)
* React 19
* TypeScript
* Tailwind CSS v4

## Installation

### Prerequisites

Ensure you have the following prerequisites installed and set up:

* Node.js 18.x or later (recommended to use Node.js 20.x or later)

1. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

   > Use `--legacy-peer-deps` flag if you face peer-dependency error during installation.

2. Start the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```
