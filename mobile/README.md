# Frontend (Mobile) - Ride Booking App

## Công nghệ sử dụng

- **React Native**: Xây dựng ứng dụng di động đa nền tảng bằng JavaScript.
- **Expo**: Công cụ hỗ trợ phát triển, build, test app React Native.
- **React Navigation**: Điều hướng giữa các màn hình.
- **Axios**: Gọi API tới backend.
- **Socket.IO Client**: Kết nối realtime với backend.
